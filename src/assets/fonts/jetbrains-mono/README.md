JetBrains Mono webfonts are vendored from the official JetBrains Mono repository:

- https://github.com/JetBrains/JetBrainsMono/tree/master/fonts/webfonts

Included weights in this directory:

- `JetBrainsMono-Regular.woff2` for `font-weight: 400`
- `JetBrainsMono-Medium.woff2` for `font-weight: 500`
- `JetBrainsMono-SemiBold.woff2` for `font-weight: 600`
- `JetBrainsMono-Bold.woff2` for `font-weight: 700`
